Strings ko reverse karne ke liye loop ka istemal karna hai. Yeh approach string ke characters ko last se pehle tak access karne ki facility deti hai.
Do strings ko anagram check karne ke liye dono strings ko sort karna hai. Agar dono sorted strings same hain, toh woh anagrams hain. Strings ko sort karne ke liye split, sort, aur join methods ka istemal hota hai. ya toh character frequencies count kar sakte hain, ya phir dono strings ko sort karke compare kar sakte hain.
Palindrome main string ko ulta kar ke check karna hota hai ke kya original aur reversed string same hain ya nahi. Loops ke through ya built-in methods ke zariye string ko reverse kar ke comparison kar sakte hain.
